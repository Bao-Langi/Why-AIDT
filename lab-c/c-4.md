# **Лабораторная работа 4**
## **Продвинутая работа с функциями (Аналоги)**

### **Задача 1** - Операции с матрицами

**Версия 1:** Реализуйте умножение матриц с использованием VLA.

**Версия 2:** Реализуйте вычисление определителя матрицы 3x3.

### **Задача 2** - Парсинг JSON

**Версия 1:** Напишите функцию парсинга простого JSON массива чисел.

**Версия 2:** Реализуйте парсинг JSON объекта с вложенными объектами.

### **Задача 3** - Системы линейных уравнений

**Версия 1:** Реализуйте решение СЛАУ методом Крамера для матриц 2x2.

**Версия 2:** Реализуйте метод Гаусса с выбором главного элемента.

### **Задача 4** - Бинарные деревья

**Версия 1:** Реализуйте АВЛ-дерево с функциями балансировки.

**Версия 2:** Реализуйте обход дерева в ширину (BFS).

### **Задача 5** - Чтение веб-страниц

**Версия 1:** Реализуйте загрузку HTML страницы и извлечение всех ссылок.

**Версия 2:** Реализуйте парсинг RSS-ленты с выводом заголовков.

### **Задача 6** - Сортировка и поиск

**Версия 1:** Реализуйте сортировку структур `Book` по названию и году издания.

**Версия 2:** Реализуйте бинарный поиск в отсортированном массиве структур `Product` по цене.

### **Задача 7** - Кэширование вычислений

**Версия 1:** Реализуйте кэширование факториала с memoization.

**Версия 2:** Реализуйте кэширование степеней двойки.

### **Задача 8** - Частота символов

**Версия 1:** Реализуйте подсчет частоты слов в тексте.

**Версия 2:** Реализуйте построение гистограммы для частоты букв.

### **Задача 9** - Упаковка данных

**Версия 1:** Реализуйте упаковку массива структур `Point` в бинарный файл.

**Версия 2:** Реализуйте сжатие данных с помощью RLE перед записью в файл.

### **Задача 10** - Генерация изображений

**Версия 1:** Реализуйте генерацию шахматной доски в PPM формате.

**Версия 2:** Реализуйте генерацию фрактала Мандельброта.
