# **Лабораторная работа 5**
## **Файловый ввод-вывод**

### **Задача 1** - Чтение файла

**Версия 1:** Напишите программу, которая читает и выводит файл задом наперед (последняя строка первой).

**Версия 2:** Реализуйте чтение файла с пропуском строк, начинающихся с "#" (комментарии).

### **Задача 2** - Запись в файл

**Версия 1:** Реализуйте запись лога с временными метками в формате "\[HH:MM:SS] сообщение".

**Версия 2:** Реализуйте запись в CSV файл данных из массива структур.

### **Задача 3** - Копирование файлов

**Версия 1:** Реализуйте копирование только строк, содержащих определенное слово.

**Версия 2:** Реализуйте копирование файла с прогресс-баром в консоли.

### **Задача 4** - Статистика файла

**Версия 1:** Реализуйте подсчет частоты каждого слова в файле.

**Версия 2:** Реализуйте поиск самого длинного слова в файле.

### **Задача 5** - Бинарные файлы структур

**Версия 1:** Реализуйте запись/чтение массива структур `Employee` с полями name, id, salary.

**Версия 2:** Реализуйте поиск структуры по id в бинарном файле.

### **Задача 6** - Произвольный доступ

**Версия 1:** Реализуйте удаление записи из бинарного файла путем пометки флагом.

**Версия 2:** Реализуйте вставку новой записи в произвольную позицию файла.

### **Задача 7** - Временные файлы

**Версия 1:** Реализуйте сортировку чисел, используя временный файл для промежуточных результатов.

**Версия 2:** Реализуйте слияние двух отсортированных файлов через временный файл.
