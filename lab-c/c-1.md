# **Лабораторная работа 1**
## Указатели 

### **Задача 1** - Работа с тройным указателем

**Версия 1:** Создайте программу, которая использует указатель типа `int***` для доступа и изменения значения целочисленной переменной. Выделите память динамически под все промежуточные указатели. Не используйте промежуточные переменные кроме `int*** ptr`.\

**Версия 2:** Реализуйте программу с использованием `char****` для доступа к символьной переменной, записывая в нее строку "Hello" посимвольно через цепочку разыменований.

### **Задача 2** - Итерация массива указателями

**Версия 1:** Создайте динамический массив из 10 целых чисел, заполните его случайными числами от 1 до 100, и выведите все элементы, используя только арифметику указателей и оператор `++`.\

**Версия 2:** Создайте массив из 15 чисел с плавающей точкой, введите значения с клавиатуры, и найдите сумму элементов, используя только указатели и оператор `--` для обхода с конца массива.

### **Задача 3** - Динамические 2D массивы

**Версия 1:** Реализуйте выделение памяти для матрицы 5x7 двумя способами: классическим (массив указателей) и непрерывным. Заполните матрицу значениями `i+j` и выведите её. Освободите память.\

**Версия 2:** Создайте матрицу NxM (N и M вводятся), выделив память одним блоком. Заполните матрицу единицами на главной диагонали и нулями в остальных элементах. Реализуйте функцию транспонирования.

### **Задача 4** - Обмен значений

**Версия 1:** Напишите функцию `swap_double(double* a, double* b)`, которая меняет местами два числа с плавающей точкой.\

**Версия 2:** Напишите функцию `swap_strings(char** s1, char** s2)`, которая меняет местами указатели на строки.

### **Задача 5** - Поиск максимального элемента

**Версия 1:** Напишите функцию `int* find_max(int* arr, int size)`, возвращающую указатель на максимальный элемент в массиве целых чисел.\

**Версия 2:** Напишите функцию `float* find_min(float* arr, int size)`, возвращающую указатель на минимальный элемент в массиве float.

### **Задача 6** - Реверс массива

**Версия 1:** Реализуйте функцию `reverse_int_array(int* start, int* end)`, которая переворачивает часть массива между двумя указателями.\

**Версия 2:** Напишите функцию `reverse_char_array(char* str)`, которая переворачивает строку на месте без использования дополнительного массива.

### **Задача 7** - Поиск подстроки

**Версия 1:** Напишите функцию `char* find_substring(char* str, char* substr)`, которая возвращает указатель на первое вхождение подстроки или NULL.\

**Версия 2:** Реализуйте функцию поиска последнего вхождения подстроки в строку.

### **Задача 8** - Копирование строки

**Версия 1:** Напишите функцию `my_strcpy(char* dest, const char* src)`, копирующую строку src в dest.\

**Версия 2:** Реализуйте функцию `my_strncpy(char* dest, const char* src, size_t n)`, копирующую не более n символов.

### **Задача 9** - Подсчет символов

**Версия 1:** Напишите функцию `count_spaces(const char* str)`, возвращающую количество пробелов в строке.\

**Версия 2:** Реализуйте функцию `count_digits(const char* str)`, подсчитывающую количество цифр в строке.


### **Задача 10** - Указатели на функции

**Версия 1:** Реализуйте функцию сортировки массива строк (char\*\[]) с использованием указателя на функцию сравнения.\

**Версия 2:** Создайте калькулятор, где операции (+, -, \*, /) реализованы как функции, а указатель на функцию выбирается через switch-case.
