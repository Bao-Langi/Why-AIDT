# **Лабораторная работа 2**
## Структуры

### **Задача 1** - Создать структуру с указателем на функцию

**Версия 1:** Создайте структуру `Calculator` с полем `int (*operation)(int, int)`. Реализуйте функции для сложения, вычитания, умножения и деления. Вызовите нужную функцию через имя переменной структуры и поле указателя на функцию.

**Версия 2:** Создайте структуру `Printer` с полем `void (*print)(void*)` и реализуйте вывод разных типов данных (int, float, char\*) через единый интерфейс.

### **Задача 2** - Структура для вектора в трёхмерном пространстве

**Версия 1:** Реализуйте структуру для вектора в 3D пространстве и добавьте функции для скалярного умножения векторов, векторного произведения и вычисления модуля вектора.

**Версия 2:** Расширьте структуру Vector3D, добавив операции нормализации вектора, умножения на скаляр и вычисления угла между двумя векторами.

### **Задача 3** - Вычисление комплексной экспоненты

**Версия 1:** Используя структуру для представления комплексного числа, вычислите комплексную экспоненту $e^z$ для числа $z∈C$ с использованием ряда Тейлора.

**Версия 2:** Расширьте функционал работы с комплексными числами, добавив функции для вычисления комплексного сопряжения, деления комплексных чисел и вычисления аргумента комплексного числа.

### **Задача 4** - Структура с битовыми полями для даты

**Версия 1:** Используя битовые поля в структуре C, создайте структуру для хранения даты (день, месяц, год) с минимальным использованием памяти.

**Версия 2:** Создайте структуру `Time` с битовыми полями для хранения времени (часы, минуты, секунды) и добавьте функции для форматированного вывода и сравнения времени.

### **Задача 5** - Реализация двусвязного списка

**Версия 1:** Реализуйте структуру и функции для создания и наполнения двусвязного списка, а также функции для его обхода и распечатки в прямом и обратном порядке.

**Версия 2:** Реализуйте двусвязный список для хранения произвольных данных, используя указатели void\* на данные в узлах списка. Добавьте функции для поиска, удаления и сортировки элементов.
