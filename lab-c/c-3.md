# **Лабораторная работа 3**
## **Объединения и перечисления (Аналоги)**

### **Задача 1** - Указатель на объединение

**Версия 1:** Создайте объединение `Value`, хранящее int, float, char. Используйте указатель для инициализации и вывода.

**Версия 2:** Создайте объединение `Color`, хранящее цвет в форматах RGB и HSV, и выведите значения через указатель.

### **Задача 2** - Побайтовый вывод

**Версия 1:** Используйте объединение для побайтового вывода значения `double`.

**Версия 2:** Используйте объединение для побайтового вывода значения `unsigned long long`.

### **Задача 3** - Перечисления

**Версия 1:** Создайте перечисление `Planets` с планетами Солнечной системы и выведите их числовые значения.

**Версия 2:** Создайте перечисление `FileTypes` для типов файлов (TXT, PDF, EXE) и выведите их коды.

### **Задача 4** - Размеченное объединение

**Версия 1:** Создайте структуру `Variant` с объединением (int, float, char\[20]) и перечислением-тегом.

**Версия 2:** Реализуйте массив таких структур с функцией поиска по типу данных.

### **Задача 5** - Хранение данных о студентах

**Версия 1:** Создайте структуру `Student` с объединением для возраста (int или строка "young"/"old").

**Версия 2:** Создайте структуру `Employee` с объединением для зарплаты (int или float).

### **Задача 6** - Управление состояниями

**Версия 1:** Реализуйте конечный автомат для светофора (RED, YELLOW, GREEN) с перечислением.

**Версия 2:** Реализуйте состояния загрузки файла (IDLE, LOADING, SUCCESS, ERROR) с перечислением.

### **Задача 7** - Битовые поля для метеоданных

**Версия 1:** Создайте структуру с битовыми полями для температуры (-50..50) и влажности (0..100%).

**Версия 2:** Добавьте поле для давления (0..1100 hPa) с минимальным использованием бит.

### **Задача 8** - Ввод/вывод через enum и union

**Версия 1:** Реализуйте программу, позволяющую вводить данные как int или double и сохранять в размеченном объединении.

**Версия 2:** Реализуйте программу для ввода геометрических фигур (круг, квадрат) с разными параметрами через объединение.
