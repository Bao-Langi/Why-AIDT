### Задача 6: "Генератор паролей"

Сгенерировать случайный пароль по правилам:

* Длина: случайно от 8 до 12 символов
* Обязательно: 2 заглавные буквы, 3 строчные, 2 цифры, 1 спецсимвол (!@#$%^&\*)
* Не должно быть повторяющихся символов подряд
* Первый символ - буква

**Реализовать:**

1. Генерацию одного пароля
2. Генерацию N паролей (пользователь вводит N)
3. Проверку сгенерированного пароля на соответствие правилам
4. Оценку сложности пароля по шкале от 1 до 10

**Критерии сложности:**

* +1 за каждую категорию символов
* +1 за длину >10
* +1 за отсутствие повторений
* +1 за спецсимволы в середине (не в начале/конце)

**Пример вывода:**

```
Сколько паролей сгенерировать? 3

Пароль 1: Ab3cD!eF9g (10 символов)
```

### Задача 7: "Шифратор сообщений"

**Вход:** строка (массив символов)
**Шаги шифрования:**

1. Шифрование методом "реверс по группам": разбить на группы по 4 символа и перевернуть каждую группу
2. Сдвиг каждого символа на его позицию в алфавите (a→+1, b→+2, ...)
3. Замена гласных букв на числа (а→1, е→2, и→3, о→4, у→5)
4. Добавление контрольной суммы (сумма кодов символов % 10)

**Пример:**

```
Исходное:  "Hello World"
Шаг 1:     "lleHoW rolld"  (разбиваем: Hell o Wo rld → lleH o oW dlr)
Шаг 2:     "mmfIpX spmmd"  (сдвиг: H+8, e+5, l+12, ...)
Шаг 3:     "mmf3pX spmmd"  (замена e→2)
Контр.сумма: 5
Результат: "mmf3pX spmmd[5]"
```

**Требуется:**

1. Ввод строки с клавиатуры
2. Вывод всех промежуточных результатов
3. Дешифратор (обратное преобразование)
4. Обработка русских и английских букв

```
S 1 0 0 1 0
```
